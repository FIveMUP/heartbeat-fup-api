## Getting Started

### Prerequisites

- Rust
- Mariadb Server
- Proxies

```sh
    cargo run
```

### Installation

Installation command

```sh
    git clone https://github.com/Apolo-sys/heartbeat-fup-api
```

You need a .env to run the project. This is an example of it:

```env
    DATABASE_URL=
```
